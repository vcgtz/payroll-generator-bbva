Dispersión de Nómina BBVA

Instrucciones de Uso

1. Cómo usar la aplicación

  1. Haz clic en Cargar CSV para seleccionar tu archivo con los datos de nómina.
  2. Verifica los datos en la tabla que se muestra.
  3. Usa Vista Previa para revisar el archivo formateado antes de generarlo.
  4. Haz clic en Generar Archivo TXT para guardar el archivo. Se te pedirá elegir la ubicación y nombre del archivo.

2. Formato del archivo CSV esperado

El archivo CSV debe tener 4 columnas en el siguiente orden (los nombres de las columnas no importan, solo el orden):

Columna Descripción Ejemplo
1. NombreNombre completo del trabajadorGARCÍA LÓPEZ MARÍA
2. ImporteMonto a pagar (puede incluir $, comas y espacios)$4,652.01
3. CuentaNúmero de cuenta BBVA (10 dígitos)0123456789
4. RFCRFC del beneficiario (opcional, puede estar vacío)GALM850101AB1

Ejemplo de contenido CSV:

Nombre,Importe,Cuenta,RFC
GARCÍA LÓPEZ MARÍA,$4652.01,0123456789,GALM850101AB1
PEÑA NIÑO JOSÉ,$12589.00,9901093870,

3. Formato del archivo TXT generado

El archivo generado sigue el layout de BBVA para dispersión de nómina. Cada línea tiene exactamente 108 caracteres con los siguientes campos:

Posición Campo Largo Descripción
1-9Secuencia9Consecutivo automático (desde 000000001)
10-25RFC16RFC del beneficiario o espacios en blanco
26-27Tipo de cuenta2Siempre 99 (Nómina)
28-47Número de cuenta20Cuenta rellenada con espacios
48-62Importe15Pesos y centavos sin punto decimal, con ceros a la izquierda
63-102Nombre40Nombre rellenado con espacios
103-105Banco destino3Siempre 001
106-108Plaza destino3Siempre 001
